Packageefme.core.graphics2d
Classpublic class Image

Class which represents an image.

The class handles both regular and tiled images. By specifying a size in the constructor, you are setting an explicit tile size.

A call to draw(screen, ...) will draw the whole image, regardless if it's tiled or not. Meanwhile, a call to drawTile(screen, tileX, tileY, ...) will draw a particular (or the whole image if untiled).
